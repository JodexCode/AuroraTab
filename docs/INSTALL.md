# 安装指南

> 最后修改时间：2026年3月28日

本文档提供 Aurora Tab 的安装和使用指南。

## 下载安装

### 方式一：直接下载（推荐）

前往 [Releases 页面](https://github.com/JodexCode/AuroraTab/releases) 下载最新版本：

| 文件格式 | 适用浏览器 | 说明 |
| :------- | :--------- | :---- |
| **.zip** | Chrome / Edge | 推荐，最稳定的方式 |
| **.crx** | Chrome / 开发者版 | 可能有限制 |
| **.xpi** | Firefox | 需要 Firefox Nightly |

### 方式二：开发者模式安装

1. 下载 `.zip` 文件并解压
2. 打开浏览器扩展管理页面：
   - Chrome/Edge: `chrome://extensions`
   - Firefox: `about:addons`
3. 开启「开发者模式」（Chrome/Edge）或「调试模式」（Firefox）
4. 点击「加载已解压的扩展程序」
5. 选择解压后的文件夹

## 首次使用

### 设置搜索引擎

1. 点击工具栏的搜索引擎图标
2. 选择常用的搜索引擎（Bing/百度/Google）
3. 选择后会记住，下次打开仍然是这个引擎

### 设置语言

1. 点击工具栏的地球图标
2. 选择「中文」或「English」
3. 立即生效

### 更换壁纸

1. 点击工具栏的图片图标
2. 选择预设渐变壁纸
3. 或点击「+」上传自定义图片/视频

### 自定义外观

1. 点击工具栏的调色板图标
2. 可调整：
   - 工具栏位置（左侧/上方）
   - 搜索栏宽度
   - 搜索栏圆角
   - 面板圆角

## 常见问题

### Q: 壁纸显示不出来？

1. 检查是否上传了过大的图片（建议小于 10MB）
2. 尝试重新选择壁纸
3. 清除浏览器缓存后重试

### Q: 如何卸载？

在浏览器扩展管理页面找到 Aurora Tab，点击移除即可。

### Q: 出现问题了怎么办？

1. 尝试禁用再重新启用扩展
2. 清除扩展数据：扩展图标右键 → 选项 → 清除数据
3. 在 [GitHub Issues](https://github.com/JodexCode/AuroraTab/issues) 报告问题

## 隐私说明

Aurora Tab 是一个本地优先的浏览器扩展：

- ✅ 所有数据存储在本地（IndexedDB）
- ✅ 不收集任何用户信息
- ✅ 不联网（搜索请求直接发送到搜索引擎）
- ✅ 开源透明
